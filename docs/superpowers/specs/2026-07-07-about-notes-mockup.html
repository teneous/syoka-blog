设计说明 · 本区块不属于页面内容 · v3 弃用宋体 + 精致细节

  1. 不改结构(承 v2):About 保持 AuthorLayout 原有的"头像卡 + 正文卡"两栏网格,`<ul>` 项目符号列表原样保留;Notes 保持原有卡片式列表(每篇一张卡)。
  2. 弃用宋体(v3 核心修正):中文语境下衬线大字容易显"呆板"而不是高级。改用加粗无衬线(font-weight 800)+ 收紧字距(letter-spacing -0.02~-0.03em),页面大标题("关于" / "Notes")叠加深浅渐变描边(与首页 H1 同款处理),质感更"科技感"而不是"文绉绉"。
  3. 找回精致感(针对"太素"的修正):卡片顶部加 2px 紫→青渐变细线;头像加圆锥渐变光环描边;小节标题("简介""关注方向"等)前加一个 8px 方形色块替代原来空荡的纯文字;page-kicker(等宽小标签 + 呼吸点)呼应首页 Hero 的 eyebrow 写法,弱化"两页各自秃头"的观感。
  4. Notes 卡片:标题改粗体无衬线,日期改等宽字体 + 主题色高亮,标签胶囊化(仍在卡片内,不改变卡片式结构);hover 只做描边变色 + 标题变色 + 箭头轻移。
  5. 色彩纪律不变:暗色靛蓝 #818cf8、浅色紫 #6d5ae0,新增青色 #22d3ee(暗)/ #06b6d4(浅)作为渐变第二色,与首页 Pipeline 流光配色同源。
  6. 本 mockup 里两页上下堆叠仅为方便你一次看完,实际是两个独立路由,互不导航拼接。
